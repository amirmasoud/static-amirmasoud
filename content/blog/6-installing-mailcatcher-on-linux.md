---
title: Installing Mailcatcher on Linux
tags: DevOps Tips
createdAt: June 13, 2020
---

1. Install essential packages:

```bash
sudo apt install gem ruby build-essential ruby-dev libsqlite3-dev
```

2. Install Mailcatcher

```bash
gem install mailcatcher
```
